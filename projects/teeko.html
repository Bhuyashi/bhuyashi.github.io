AI Teeko Game Player

March 2025 | Minimax ML

Project Overview

Developed an intelligent AI game player for Teeko, a strategy game played on a 5x5 board. The project involved implementing the minimax algorithm with depth-limited search, heuristic evaluation, and custom scoring functions to simulate strategic gameplay.

About Teeko

Teeko is a two-player abstract strategy game invented by John Scarne. The objective is to arrange four of your pieces in a row, column, diagonal, or in a square on a 5x5 board. The game has two phases: the drop phase (placing pieces) and the move phase (moving pieces to adjacent spaces).

AI Implementation

  • Minimax Algorithm: The AI uses a depth-limited minimax search to evaluate possible moves and select the optimal strategy.
  • Heuristic Evaluation: Custom scoring functions estimate the value of board states, considering potential wins, blocks, and strategic positions.
  • Game Logic: Handles both the drop and move phases, checks for win conditions, and simulates intelligent gameplay.

How to Use

  1. Clone the GitHub repository.
  2. Ensure you have Python installed.
  3. Run game.py to play against the AI or test the AI's performance.
